## Installation

1. Clone the repository:
```bash
git clone https://github.com/Alston-Choong/motimax.git
cd motimax
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env
```

4. Start the development server:
```bash
npm run dev
```
